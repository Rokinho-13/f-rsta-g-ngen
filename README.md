## Bidra till projektet

### Git Workflow

När du arbetar med projektet, följ dessa steg:

1. **Skapa en ny branch** för dina ändringar:
   ```bash
   git checkout -b claude/din-feature-namn-sessionId
   ```

2. **Gör dina ändringar** och committa regelbundet:
   ```bash
   git add .
   git commit -m "Beskrivande commit-meddelande"
   ```

3. **Pusha dina ändringar**:
   ```bash
   git push -u origin din-branch-namn
   ```

4. **Skapa en Pull Request** på GitHub för att granska dina ändringar

### Commit-meddelanden

Skriv tydliga och beskrivande commit-meddelanden som förklarar:
- Vad som ändrades
- Varför ändringen gjordes
- Eventuella viktiga detaljer
